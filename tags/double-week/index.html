    5323. 根据数字二进制下 1 的数目排序(Easy)

    Problem Link

    https://leetcode-cn.com/problems/sort-integers-by-the-number-of-1-bits/

    Description

    给你一个整数数组 arr 。请你将数组中的元素按照其二进制表示中数字 1 的数目升序排序。

    如果存在多个数字二进制中 1 的数目相同，则必须将它们按照数值大小升序排列。

    请你返回排序后的数组。
